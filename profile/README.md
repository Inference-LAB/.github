# [INFERENCE Lab](https://www.inference-lab.org)

Applied AI engineering lab based in Multan, Pakistan.

We train engineers to build and deploy real AI systems —
not theory, not tutorials, not certificates.

## What We Build
- End-to-end AI applications
- LLM and RAG pipelines
- Speech AI systems
- NLP research tools
- MLOps infrastructure

  - RUEmoCorp: A Large-Scale Roman Urdu Emotion Corpus with Cross-Institute Annotation Validation and State-of-the-Art Emotion Classification. [Read here](https://doi.org/10.21203/rs.3.rs-9759243/v1)
